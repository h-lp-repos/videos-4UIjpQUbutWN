# Lesson L3: Multi-Agent Systems

This repository contains materials for Lesson L3: Multi-Agent Systems, including scripts, storyboards, slide deck outlines, and code examples for three videos:

- Video 1: Introduction & Motivation — Why Multi-Agent Systems?
- Video 2: Technical Deep-Dive — Architectures & Collaboration Patterns
- Video 3: Hands-On Integrative Demo — Building a Multi-Agent Query Router

## Repository Structure

```
lesson-l3-multi-agent-systems/
├── video-1-introduction-motivation/
├── video-2-technical-deep-dive/
└── video-3-hands-on-integrative-demo/
    └── code-examples/
        └── data/
```

## Setup

Install dependencies:

```bash
pip install -r requirements.txt
```

Optionally, launch Jupyter Lab:

```bash
jupyter lab
```
