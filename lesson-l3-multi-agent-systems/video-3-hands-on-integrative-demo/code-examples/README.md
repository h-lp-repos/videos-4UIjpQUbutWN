# Code Examples

This folder contains starter code for Video 3: Multi-Agent Query Router demo.

## Files

- `orchestrator.py`: Orchestrator Agent script to route queries.
- `hr_agent.py`: HR Agent implementation using CSV knowledge base.
- `tech_agent.py`: Tech Agent implementation using CSV knowledge base.
- `data/hr-kb.csv`: Sample HR knowledge base.
- `data/tech-kb.csv`: Sample Tech knowledge base.

## Usage

```bash
cd lesson-l3-multi-agent-systems/video-3-hands-on-integrative-demo/code-examples
pip install -r ../../../../requirements.txt
python orchestrator.py
```
